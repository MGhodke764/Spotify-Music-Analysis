# Spotify-Music-Analysis
Developed an interactive Spotify Music Analysis Dashboard using Microsoft Power BI to analyze streaming patterns, track performance, artist popularity, and music attributes from Spotify's most-streamed songs dataset. The dashboard provides meaningful insights through dynamic visualizations, KPIs, and interactive filters.

## Project Objectives
-Analyze Spotify's most streamed tracks and artists.

-Identify streaming trends across years and months.

-Compare track performance based on release date and popularity.

-Explore music characteristics such as energy, danceability, speechiness, and instrumentalness.

-Create an interactive dashboard for data-driven music analysis.

## Tools & Technologies
-Microsoft Power BI

-Power Query Editor

-DAX (Data Analysis Expressions)

-Data Visualization

-Data Cleaning & Transformation

## Dashboard Features

1-Total Streams, Total Tracks, and Average Streams KPI cards.

2-Interactive slicers for filtering artists and tracks.

3-Streams analysis by release year.

4-Monthly tracks and average streams analysis.

5-Daily streaming trends visualization.

6-Most streamed track details with album artwork.

7-Top 5 most streamed tracks comparison.

8-Music attribute analysis including:

Energy

Danceability

Speechiness

Liveness

Instrumentalness
